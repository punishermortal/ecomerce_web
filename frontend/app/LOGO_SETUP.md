# Quick Logo Setup

## Step 1: Add Logo File

Place your NEX BLOOMS logo image in:
```
frontend/public/logo.png
```

## Step 2: Restart Server

Restart your Next.js development server:
```bash
npm run dev
```

## Step 3: Verify

The logo should now appear:
- ✅ Navbar (top left, circular)
- ✅ Footer (circular)
- ✅ Homepage hero (large circular with border)

## Logo Requirements

- **Format**: PNG, JPG, or SVG
- **Size**: 512x512px or larger (square)
- **Background**: Transparent (preferred) or solid color
- **Name**: `logo.png` (or update components if using different name)

The logo will be automatically displayed in a circular format with proper styling!
